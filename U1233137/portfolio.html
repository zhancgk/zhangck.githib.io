💻 響應式雲端個人記帳系統

結合前端開發、資料庫管理與財務分析的完整資訊系統專案

HTML5 Bootstrap 5 JavaScript MySQL

專案介紹

本系統專為學生與社會新鮮人打造， 提供簡潔直覺的財務管理介面， 協助使用者記錄日常收支、 分析消費習慣並建立理財觀念。

🎯 開發動機

許多人無法有效掌握自己的財務狀況， 因此我希望透過記帳系統協助使用者建立良好的理財習慣， 同時提升自己在前端設計與資料庫管理方面的能力。

💡 開發心得

透過本次專案實作， 我學習到 Bootstrap 響應式網頁設計、 JavaScript 表單驗證以及資料庫規劃的重要性。 在開發過程中最大的挑戰是介面設計與資料結構規劃， 透過不斷測試與修正， 逐步提升系統的完整度與使用體驗。
